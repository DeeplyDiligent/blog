Fighting for Completion

Jul 25, 2018

When I was much younger and struggled to get my homework done on its due date, I would become terrified. We know that every single task is like a small mountain that we need to climb and that theres a drop at the end where the grass will be a lot greener.

Conquering Mount Everest

Iâ€™m sure you have a task with a high importance, a task that will make a crucial impact on your life, that is currently too big for you to do. Two possible issues are that you may have many different smaller tasks which are more urgent (have a closer due date) or you have no clue where to start the big task. This is your Mount Everest. Lets take you through conquering it

Getting to the bottom

Before we can start climbing we must push all other tasks aside, leaving just the mountain itself. This is through prioritisation. Grab a pencil and start batching all the smaller tasks, then complete them one after another with minimal time wasted. Any tasks that you can defer, do it!

Starting the climb

Whether you cant think of somewhere to begin, or cant muster the motivation to put pen to paper, I guarantee that if you:

  • Think of a tiny part of the task (read the assignment, write the first paragraph etc)
  • Allocate a set time that you think will be enough for it

You will find the task at least partially interesting and doable. That initial hump is the biggest. A rule in physics is that a system is much easier to keep going than it is to restart. I believe this principle applies nicely here as well. All the best!
