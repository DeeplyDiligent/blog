My Life on Kaizen (Part 5)

Aug 9, 2018

Catching up is my life. It is only because I keep track of all the tasks which need to be done, date them, and do them after they are overdue that I can keep up with the constant gush of tasks that come my way.

Takling Overdue Tasks

We’ve all been there. You’ve had a day off and are ready to kick back into the rhythm of your daily routine. The problem is that all the tasks from yesterday are now overdue. We need a proven strategy, an algorithm, to tackle all of this.

My Algorithm

My algorithm for tackling weekly work

  • slides -> onenote (1h 15min)
  • make a speech using onenote notes
  • quizlet

When I Am getting Behind

  • don’t do it again, you have a set number of days
  • remove the distraction
  • do a five minute mindfulness session
  • start again from where you left off, do not skip anything
